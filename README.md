# GameLib Web App

- A web app that showcases some of the released game all the way from AAA to Indie category. Viewed as a library of "Games".

## Planned Features

- Genre Selection
  - Allows viewing games based on selected filter
- Add Games
  - User uploade game metadata and thumbnail

## Technology used

- React
- TypeScript
- Vite
- MongoDB
- ExpressJS
- Node.js
